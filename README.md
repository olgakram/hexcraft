# hexcraft

## Разработка

### Первый запуск
Для удобной работы предварительно установите bower, sequelize-cli и gulp глобально

```
npm i -g bower
npm i -g sequelize-cli
npm i -g gulp
```

Затем установите зависимости проекта и начинайте работать

```
npm i
npm start
```
